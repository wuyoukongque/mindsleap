UI & Site

完整营销页面 mockup + 单个 UI 组件 preview。需要复用？复制 preview/ 下的 HTML，或者用 ui_kits/website/ 下的 React 组件。

03 · UI & Site
Quick start · 怎么用 html · jsx · next.js

复用单个组件

每个 preview 是独立 HTML/CSS，直接抄：

// 1. 打开你要的 preview，例如
preview/component-news-card.html

// 2. 复制 <style> 里的 CSS 和 <body> 里的 HTML
// 3. 确保新文件引用了 tokens：
<link rel="stylesheet" href="colors_and_type.css">

所有组件都基于 colors_and_type.css 的 tokens，主色/字体直接生效。

用在生产网站

生产代码在 site/src/（Next.js 16 App Router）。UI kit 里的 React 组件用 Babel standalone 渲染，是给设计参考用的，不是生产代码：

  • UI kit JSX：ui_kits/website/*.jsx
  • 生产 React：site/src/components/
  • 页面路由：site/src/app/[locale]/

新组件先在 UI kit 里设计验证，再翻译到生产。

Brand

颜色、字体、间距、阴影、Logo——品牌的基本视觉语言。所有 token 都定义在 colors_and_type.css。

01 · Brand
Quick start · 怎么用 copy · paste · use

颜色 + 字体

/* 一行 import，全部 tokens 到手 */
@import "colors_and_type.css";

color: var(--ml-blue);          /* #1e477c · 主色 */
color: var(--ml-blue-light);    /* #2a5a9e · 次色 */
color: var(--ml-blue-accent);   /* #3b82f6 · 链接/强调 */

font-family: var(--font-sans);  /* 正文 · SF / Inter */
font-family: var(--font-sans-zh); /* 中文优先栈 */

完整 token 列表见 colors_and_type.css（51 个 token）。

Logo + 图标

  • Logo PNG：assets/logos/（blue / white / joint）
  • 双语 logo 排版：见下方 Logos 区块
  • 图标 SVG：assets/icons/（2px stroke · Heroicons 风格）
  • 品牌字体：fonts/MindsLeap-Brand.otf，仅用于 wordmark + hero

三条不能违反的规则

  • 主色只有 #1e477c，accent 只有 #3b82f6
  • UI 里禁止 emoji——用 assets/icons/ 里的 SVG
  • 品牌名永远中英成对：MindsLeap 心智悦动

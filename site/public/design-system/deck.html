Deck templates

1920×1080 品牌 deck，4 种主题预设。点击进入后，右下角的 Tweaks 面板可继续切换字体（Brand/Inter/Noto）与密度（松散/标准/紧凑）。

02 · Deck
Quick start · 怎么做新 deck copy · edit · share

从模板开始

  1. 下方点任意主题卡片 → 进入 deck
  2. 右下角 Tweaks 面板继续切字体 / 密度
  3. 要做新内容：复制 deck-template/ 整个文件夹
  4. 编辑 HTML 里的 <section class="slide">

键盘快捷键

  • ← / → 翻页 · Home / End 首末
  • R 回首页 · 1–9 跳页
  • 浏览器 Print → Save as PDF 导出

URL 参数

# 同一个 HTML，URL 决定初始视觉
?theme=light    # 浅 / brand / dark / magazine
?heading=brand  # brand / inter / noto
?density=standard # spacious / standard / compact

# 组合用 &
deck.html?theme=dark&heading=inter&density=compact

把链接复制给同事，他打开就是同样的预设。

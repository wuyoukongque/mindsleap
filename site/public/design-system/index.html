Working with this system human + AI

给团队成员

每个分类页里都有可直接复制的 token 和 snippet。最常用的几个：

/* 在 CSS 里 */
@import "site/public/design-system/colors_and_type.css";

color: var(--ml-blue);            /* #1e477c — 品牌主色 */
font-family: var(--font-heading); /* 标题字体 */

所有 token 定义在 colors_and_type.css。完整规则读 README.md（含语言/语调/不能做的事）。

给 AI agent

这个文件夹是一个 Claude Code skill。Agent 的入口顺序：

  • SKILL.md — 你能/不能做什么
  • README.md — 品牌叙事 + tokens + 规则
  • colors_and_type.css — 直接 import 用
  • preview/*.html — 单个组件复制粘贴

需要做视觉浏览，打开 index.html；需要写代码，读 README + 用 tokens。

Dev server：http://127.0.0.1:3000/design-system/ · 离线打开：open site/public/design-system/index.html

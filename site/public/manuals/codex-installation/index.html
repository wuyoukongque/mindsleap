Codex 安装攻略

这是一页可以直接转发给学员和同事的公开安装清单。会前把账号、电脑环境和自检跑通，现场就把时间留给真实任务。

开始安装 官方 Quickstart 返回 Manuals

先选你要用的入口

第一次上手时，先选最符合自己工作习惯的入口，不要一开始就把所有东西都装满。

推荐给大多数人

ChatGPT 桌面 App / 网页

适合先用自然语言安排任务、整理文档、做网页预览和让 Codex 帮你操作本地项目。

本地项目交付

Codex CLI

适合需要读写本地文件、跑命令、提交代码、做网站发布的人。现场如果要完整跑项目，建议至少一台电脑装好 CLI。

代码编辑器

IDE Extension

适合已经常用 VS Code 等编辑器的同事。它和 CLI 共享登录与配置，可以在编辑器里直接协作。

安装步骤

下面按现场最常见的 Mac / Windows 场景写。只要跑通登录和诊断，就算会前准备完成。

1

准备基础环境

使用 Chrome 或 Edge，确认网络可以正常访问 ChatGPT / OpenAI。需要本地交付项目的同事，提前安装 Git、Node.js LTS；Windows 同事建议同步安装 GitHub CLI。

2

安装 ChatGPT 桌面 App

打开 ChatGPT 官方网站，下载并登录桌面 App；如果暂时不装 App，也可以先用 ChatGPT 网页版。

3

安装 Codex CLI

Mac / Linux 可以在终端里运行官方安装脚本；Windows 可以在 PowerShell 里运行官方安装脚本。安装完成后，重新打开终端。

# macOS / Linux
curl -fsSL https://chatgpt.com/codex/install.sh | sh

# Windows PowerShell
irm https://chatgpt.com/codex/install.ps1 | iex
如果你已经熟悉 Node.js，也可以使用 npm 全局安装。现场建议优先按官方脚本走，少一个变量，少一点玄学。
4

登录 Codex

安装后在终端运行登录命令，浏览器会弹出 OpenAI 登录页面。请使用你准备现场使用的 ChatGPT / OpenAI 账号。

codex login
5

跑一次自检

诊断通过后，现场就可以让 Codex 进入本地项目，读文件、改页面、跑检查和协助发布。

codex doctor

现场前检查清单

如果只做一件事，就确保下面几项都完成。卡住也没关系，把错误截图带到现场，我们可以快速定位。

必须完成

  • 能打开 ChatGPT，并确认登录的是自己的账号。
  • 终端里运行 codex --version 有返回版本号。
  • 终端里运行 codex doctor 可以生成诊断结果。
  • 知道自己要让 AI 帮忙完成的一个真实任务。

常见卡点

  • 网络无法访问 OpenAI：先换稳定网络或代理环境。
  • 命令不存在：重开终端，或确认安装目录是否加入 PATH。
  • 登录跳转失败：复制终端里的链接到浏览器，或换默认浏览器重试。
  • Windows 权限问题：用普通 PowerShell 先试，不行再以管理员权限运行。

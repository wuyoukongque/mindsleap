02
PORTFOLIO DECISION

Agent 不是按 Wishlist 排队，而是用同一套标准决定是否值得投资

价值、数据、Owner、采用路径与错误成本缺一不可。

100 分投资评分

先比较真实业务价值，再用证据置信度与风险扣分校准

20%战略一致性是否直接支持平台服务化和交易闭环
20%可量化业务价值增长、人效、运营或风险收益
15%频率与痛点是否持续发生且消耗真实资源
15%数据准备度样本、字段、质量、权限和接口
10%90 天可交付是否能在真实用户中形成闭环
10%Owner 与采用谁负责、谁使用、如何改变流程
10%跨会员复用能否成为行业产品或模板
-15风险扣分责任、合规、错误成本和不可逆影响
校准得分 = 原始得分 × 证据置信系数 − 风险扣分。
MindsLeap Agent 组合评估框架。当前证据主要为单次决策者访谈与公开资料，置信系数原则上不高于 0.70。
